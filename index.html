First-Time Setup

1

Navigate to Settings

Go to Settings page and access Profile Settings, Duty Settings, Role & Permissions, and User Management.

2

Configure Organization

Add ranks, companies, leave types, absence types, and create initial user accounts with appropriate permissions.

3

Add Soldiers

Create soldier profiles with personal information, assign ranks, and organize by companies.

4

Create Duties

Set up duties and assign soldiers using fixed, roster, or rank-group assignment methods.

5

Process Requests
