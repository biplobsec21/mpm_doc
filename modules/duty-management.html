Duty Management

Create and assign duties with flexibility and automation

Module Overview

Duty Management is the operational core of personnel scheduling. Create duties, assign soldiers efficiently using three methods (Fixed, Roster, Rank-based), and rely on automatic availability checking to prevent conflicts. The system manages duty status, supports bulk operations, and enables easy export functionality.

3 Assignment Methods

Fixed, Roster, Rank Groups

Automatic Checking

Real-time availability

Bulk Operations

Efficient mass management

Key Features

1. Create and Manage Duties

Create new duties with detailed information, set dates and times, define required soldier count, and manage duty status through lifecycle.

2. Assign Soldiers (3 Methods)

Fixed Assignment: Manually select specific soldiers. Roster-based: Assign soldiers from predefined rosters. Rank Groups: Automatically select from specific rank groups.

3. Automatic Availability Checking

The system automatically checks if soldiers are available based on leave, medical status, other duties, training, and appointments. Prevents double-booking.

4. Duty Status Management

Manage duty lifecycle: Planned → Active → Completed. Track status changes, modify duty details, and record completion information.

5. Duplicate Duties

Quickly create recurring duties by duplicating existing ones. Modify the copy as needed to create similar duties efficiently.

6. Export Functionality

Export duty information to standard formats for reporting, planning, and integration with other systems.

Frequently Asked Questions

What happens if no soldiers are available?

The system will show a warning. You can either reassign duties, find alternate soldiers, or manually override availability if needed (with supervisor approval).

Can I change assigned soldiers after creation?

Yes, you can reassign soldiers as long as the duty is not marked as completed. Availability is rechecked during reassignment.

What does "rank group" assignment mean?

It automatically assigns soldiers based on their rank. For example, selecting "Officer" assigns all available officers to the duty. This is useful for position-specific duties.

Can overnight duties be split across days?

Yes, the system supports multi-day duties including overnight assignments. Availability is checked across all affected days.

Troubleshooting

Soldiers showing as unavailable when they should be available

Check if soldiers have active leave, medical status, or other conflicting duties/training. Review their status and resolve conflicts before assignment.

Cannot create duty with no soldiers assigned

At least one soldier must be assigned to a duty. If no soldiers are available, resolve availability conflicts first or adjust duty requirements.

Rank group assignment not working

Ensure the selected rank exists in your system and there are active soldiers with that rank. Create soldiers with the required rank first.
